✅ v5.x Near-Term — Implemented in v5.5.11

DONE
Server-Sent Events (SSE)
HTTP long-poll bridge for real-time data push. EventSource-compatible. Channels: market, portfolio, alerts. No extra port.
DONE
Cross-Platform File Watcher
config.pson hot-reload using inotify (Linux), ReadDirectoryChangesW (Windows), kqueue (macOS). Poll fallback included.
DONE
Binary Serialization (METI Format)
CRC-32 verified, little-endian binary format. ~10x faster than JSON. Types: double[], float[], int64[], string[], matrix, map.
DONE
Prometheus-Compatible /metrics Endpoint
Text exposition format v0.0.4. Scrapeable by Prometheus, Grafana Agent, VictoriaMetrics. Namespace: metis_genie_*.
DONE
Compile-Time Route Table (constexpr)
RouteDescriptor array with flags (PUBLIC, CACHEABLE, SSE_STREAM, etc). Route metadata available at compile time for OpenAPI generation.
DONE
gzip/deflate Response Compression
Pure C++20 store-only RFC 1951 deflate wrapped in RFC 1952 gzip. Full zlib deflate when GENIE_USE_ZLIB=1. Configurable min-size and level.

Design Principles (All Versions)

CORE
Zero External Dependencies
SQLite3 bundled. No Boost, OpenSSL, Docker, or package manager required.
CORE
PSON Configuration
All parameters in config.pson. // comments and trailing commas supported.
CORE
C++20 Throughout
constexpr, concepts, jthread, ranges, designated initializers, std::span.
CORE
Cross-Platform
Windows (MinGW/MSVC), Linux (GCC/Clang), macOS (Apple Clang). Same source.
CORE
Single Executable
One binary, no daemon dependencies, no container runtime needed.
CORE
American English
Consistent spelling throughout identifiers, comments, and documentation.
